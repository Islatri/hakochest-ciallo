创建 NPM 包: @hakochest/ciallo

1. 创建项目结构

首先创建项目目录并初始化：

mkdir ciallo-package
cd ciallo-package
npm init --scope=hakochest

2. 项目文件结构

ciallo-package/
├── package.json
├── index.js
├── README.md
└── .gitignore

3. package.json 配置

{
  "name": "@hakochest/ciallo",

    "type": "git",

4. 核心代码 (index.js)

/**
 * 输出 'ciallo' 到控制台
 * @returns {void}
 */
function ciallo() {
    console.log('ciallo');
}

module.exports = ciallo;

5. 创建 README.md

# @hakochest/ciallo

一个简单的 NPM 包，用于练习包的发布与维护。

## 安装

```bash
npm install @hakochest/ciallo
```

## 使用

```javascript
const ciallo = require('@hakochest/ciallo');

ciallo(); // 输出: ciallo
```

## API

### ciallo()

调用后在控制台输出 "ciallo"。

6. 发布到 NPM

首先登录到 npm：

npm login

然后发布包：

npm publish

注意：确保包名在 npm 上是唯一的，并且你已经登录了有权限发布到 @hakochest 命名空间的账户。

7. 使用示例

// 安装后使用示例
const ciallo = require('@hakochest/ciallo');

// 调用函数
ciallo(); // 控制台输出: "ciallo"

// 也可以直接使用
require('@hakochest/ciallo')(); // 控制台输出: "ciallo"

8. 更新版本

当你需要更新包时：

npm version patch  # 修复bug，小版本更新
npm version minor  # 新增功能，中版本更新
npm version major  # 重大变更，大版本更新

然后再次发布：

npm publish
